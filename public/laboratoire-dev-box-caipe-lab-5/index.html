Instructions

Démanteler la pile

Vous libérez les ressources : supprimez d'abord la dev box, puis démantelez toute la pile CAIPE avec IDP Builder, et confirmez que le cluster kind a bien disparu.

Progression0 / 3

Étape 1 · Supprimer la dev box

Retirez l'application ArgoCD et le namespace de la dev box alpha dans le Terminal : 
kubectl delete application dev-box-alpha -n argocd

Étape 2 · Démanteler la pile

Supprimez le cluster kind et toute la pile installée par IDP Builder : 
idpbuilder delete

Étape 3 · Vérifier la suppression

Confirmez qu'aucun cluster kind ne subsiste : 
kind get clusters

Critères de réussite

  • Dev box alpha supprimée.
  • Pile CAIPE démantelée (idpbuilder delete).
  • Aucun cluster kind restant.

Bravo — vous avez bouclé la narration self-service de référence de CAIPE !
