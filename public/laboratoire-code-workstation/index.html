Instructions

Create Evaluation Dataset

Build a small evaluation dataset for Kubernetes answers, run scoring code against it, and validate the result directly inside the workstation.

Progress0 / 4

Step 1: Create Golden Dataset

Use the Datasets tab and keep a JSON array with questions, reference answers and expected keywords.
[ { "id": "k8s-001", "question": "How do I check the logs of a Kubernetes pod?", "reference_answer": "Use kubectl logs followed by the pod name.", "expected_keywords": ["kubectl", "logs", "pod"] } ]

Step 2: Write the Scorer

In workspace/main.js, transform the dataset and print a quality report. The variable dataset is available when you run the file.

Step 3: Execute Code

Run node main.js in the terminal or click Run Code. Console output appears in the terminal.

Step 4: Validate

Run check. The lab verifies that the dataset is valid and that your scorer executed successfully.

Lab Copilot
